# 鹅厂成长伙伴 - 腾讯校园招聘陪伴平台

一个面向大学生的求职陪伴网站前端，使用 Next.js + TypeScript + Tailwind CSS 开发。

## 🎯 核心功能

### 1. 智能画像页 (`/profile`)
- 年级选择：大一至博士六个选项
- 专业选择：计算机、软件工程、电子信息等八大类
- 技能标签：JavaScript、Python、React 等 25 个技能标签，支持多选
- 兴趣方向：前端、后端、AI 等 16 个方向标签
- 动态标签切换：使用 Framer Motion 实现流畅的标签切换动画
- AI 能力分析：展示技术能力、项目经验、软技能、岗位匹配度四个维度
- **[新增] AI 技能评估模块**：嵌入评估问卷，考察学生对 AI 工具的认知，根据评分动态推荐学习资源和实习岗位

### 2. 问答科普区 (`/chat`)
- 聊天机器人界面：仿微信风格的气泡聊天界面
- 流式输出文本：使用打字机效果逐字显示 AI 回复
- 快捷问题：提供 4 个常见问题快速入口
- 智能回复：根据关键词匹配预设的 AI 回复内容
- 在线状态指示：显示 AI 助手在线状态

### 3. 岗位匹配看板 (`/match`)
- 岗位列表：前端、后端、AI、产品经理、UI设计师 五个岗位
- Echarts 雷达图：展示用户能力与岗位要求的五维雷达图
  - 技术能力
  - 框架掌握
  - 设计能力
  - 沟通能力
  - 学习能力
- 对比模式：可切换对比视图，同时显示用户能力和岗位要求
- 差距分析：直观展示各维度差距和提升建议
- 匹配度评分：每个岗位的匹配百分比
- **[新增] 岗位显微镜详情**：双击岗位卡片弹出模态框，展示日常工作拆解、成长路径（时间轴）、薪资福利透视镜

### 4. 腾讯全景地图 (`/` 首页新增 section)
- **[新增] 事业群网格布局**：展示 WXG、IEG、CSIG、PCG、CDG、TEG 六大事业群
- 每个卡片包含：事业群名称、核心产品标签（微信、王者荣耀、腾讯云等）
- 交互效果：鼠标悬停卡片轻微上浮，显示部门简介和"查看相关岗位"按钮
- 响应式设计：移动端单列、平板双列、桌面三列布局

## 🎨 设计特点

- **腾讯品牌色系**：主色 `#0052d9`(腾讯蓝)，辅助色 `#00a870`(腾讯绿)，强调色 `#ff6b35`(温暖橙)
- **卡片圆角设计**：所有卡片使用 `rounded-2xl`(16px)圆角
- **悬停动效**：卡片悬停时轻微放大(`scale(1.02)`)和阴影加深效果
- **渐变背景**：Hero 区域使用蓝绿渐变背景
- **温暖色调**：整体背景使用 `#fef9f3` 温暖色调
- **动画流畅**：使用 Framer Motion 实现页面切换、标签切换、模态框弹出等动画效果

## 🛠️ 技术栈

- **框架**: Next.js 16.2.6 (React 19)
- **语言**: TypeScript
- **样式**: Tailwind CSS 4
- **动画**: Framer Motion
- **图表**: Echarts + echarts-for-react
- **图标**: 使用 Emoji (符合简洁风格)

## 🎯 页面路由

- `/` - 首页：介绍平台功能、腾讯全景地图、统计数据
- `/profile` - 智能画像：填写个人信息生成求职画像 + AI技能评估
- `/chat` - 问答科普：与 AI 助手对话
- `/match` - 岗位匹配：查看岗位匹配度、雷达图、岗位详情

## ✨ 特色功能详解

### 1. 腾讯全景地图 (OrgChart.tsx)
- **响应式网格布局**：使用 Tailwind CSS Grid 实现移动端1列、平板2列、桌面3列
- **事业群卡片**：每个卡片包含渐变头部、核心产品标签、办公地点、员工规模
- **悬停交互**：鼠标悬停时卡片上浮(`translateY(-4px)`)，展开显示部门详情和"查看相关岗位"按钮
- **动画效果**：使用 Framer Motion 的 `whileHover` 和 `AnimatePresence` 实现流畅动画

### 2. 岗位显微镜详情 (JobDetailModal.tsx)
- **模态框设计**：点击岗位卡片弹出详情模态框，使用 `framer-motion` 实现缩放淡入动画
- **日常工作拆解**：使用编号列表展示具体职责（1-5项）
- **成长路径时间轴**：垂直时间轴展示1-5年职业发展路径
- **薪资福利透视镜**：图标化展示14-20薪结构、安居计划、免费三餐等福利

### 3. AI 技能评估 (AISkillAssessment.tsx)
- **评估问卷**：5道选择题，考察学生对 AI 工具的认知程度
- **动态推荐**：根据评分结果（0-5分小白、6-10分探索者、11-15分达人）动态推荐学习资源或实习岗位
- **进度指示**：顶部进度条显示答题进度
- **可折叠设计**：完成评估后结果持久显示，也可折叠收起

## 📝 注意事项

- 本项目为前端演示项目，AI 回复为模拟数据
- 实际部署需要连接后端 API 实现真实的 AI 对话功能
- 图片资源使用 Emoji 代替，实际项目可替换为专业图标库（如 Lucide Icons）
- 岗位详情数据仅包含前端和后端两个岗位，可扩展添加更多
