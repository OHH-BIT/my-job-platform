# 问答科普与实习专区优化总结

## 一、优化内容概述

本次优化针对"问答科普"和"实习专区"两个模块进行了核心功能增强：

### 1. 问答科普 - AI 回答文本清洗与格式化

- `src/lib/textCleaner.ts` - AI 文本清洗工具

**问题**：实习岗位详情缺少视频介绍，无法直观展示岗位内容。

**解决方案**：
- 创建 `videoParser.ts` 视频链接解析工具
- 创建 `VideoPlayer.tsx` 响应式视频播放器组件
- 在实习岗位数据模型中添加视频字段
- 在岗位详情弹窗中展示视频

**核心文件**：
- `src/lib/videoParser.ts` - 视频链接解析工具
- `src/components/VideoPlayer.tsx` - 视频播放器组件
- `src/app/intern-jobs/page.tsx` - 实习专区页面（已更新）

## 二、技术实现细节

### 1. AI 文本清洗工具 (`textCleaner.ts`)

#### 功能函数：

**`cleanAIResponse(text: string): string`**
- 去除多余的 Markdown 标记（#、*、_ 等）
- 清理转义符（\n、\"、\' 等）
- 清理无意义的占位符（[placeholder]、{{placeholder}} 等）
- 规范化换行符（多个连续换行符合并为最多两个）
- 清理特殊符号（零宽字符、&nbsp;、HTML 注释等）
- 保留核心 Markdown 格式（**、*、<code>、列表等）

**`textToHTML(text: string): string`**
- 将清洗后的文本转换为 HTML
- 支持 **加粗**、*斜体*、`代码`、• 列表、换行等格式
- 自动识别列表项并转换为 `<ul>/<li>` 结构
- 防止 XSS 攻击（转义 HTML 特殊字符）

**`processAIResponse(text: string): string`**
- 完整的 AI 回答处理流程
- 先清洗文本，再转换为 HTML
- 返回可直接插入 DOM 的 HTML 字符串

**`useProcessedAIResponse(content: string)`**
- React Hook，用于在组件中处理 AI 回答
- 返回 `{ __html: string }` 对象，可直接用于 `dangerouslySetInnerHTML`

#### 使用示例：

```typescript
import { cleanAIResponse } from '@/lib/textCleaner';

// 在组件中使用
const cleanedText = cleanAIResponse(aiResponse);
```

### 2. 视频链接解析工具 (`videoParser.ts`)

#### 功能函数：

**`parseVideoUrl(url: string): VideoInfo | null`**
- 解析视频 URL，提取平台、视频 ID、嵌入 URL 等信息
- 支持平台：
  - B站（Bilibili）：`https://www.bilibili.com/video/BV1xx411c7mD`
  - 腾讯视频：`https://v.qq.com/x/cover/xxx.html`
  - YouTube：`https://www.youtube.com/watch?v=xxx`
- 返回 `VideoInfo` 对象或 `null`（如果不是有效视频链接）

**`VideoInfo` 接口**：
```typescript
interface VideoInfo {
  platform: 'bilibili' | 'tencent' | 'youtube' | 'unknown';
  videoId: string;
  embedUrl: string;      // iframe 嵌入 URL
  thumbnailUrl?: string;  // 视频封面 URL（可选）
}
```

**`extractVideoUrls(text: string): string[]`**
- 从文本中提取所有视频链接
- 返回视频 URL 数组

**`getPlatformName(platform): string`**
- 获取平台显示名称（如"哔哩哔哩"、"腾讯视频"）

**`getPlatformIcon(platform): string`**
- 获取平台图标（Emoji）

#### 使用示例：

```typescript
import { parseVideoUrl } from '@/lib/videoParser';

const videoInfo = parseVideoUrl('https://www.bilibili.com/video/BV1xx411c7mD');
// 返回：
// {
//   platform: 'bilibili',
//   videoId: 'BV1xx411c7mD',
//   embedUrl: '//player.bilibili.com/player.html?bvid=BV1xx411c7mD&autoplay=0',
//   thumbnailUrl: '//www.bilibili.com/video/BV1xx411c7mD'
// }
```

### 3. 视频播放器组件 (`VideoPlayer.tsx`)

#### 组件特性：

**响应式设计**：
- 16:9 宽高比（使用 `paddingTop: 56.25%` 技巧）
- 宽度 100%，自适应父容器
- 移动端和桌面端完美适配

**播放体验**：
- 精美的占位图/海报（支持自定义 poster）
- 播放按钮动画（hover 放大、tap 缩小）
- 点击播放按钮后才加载 iframe（延迟加载，提升性能）
- 加载状态指示器（spinner + 文字）
- 错误处理（加载失败时显示错误信息和备用链接）

**安全特性**：
- iframe `sandbox` 属性限制（只允许同源、脚本、弹窗、表单）
- 防止 XSS 攻击

**平台支持**：
- 自动识别视频平台（B站、腾讯视频、YouTube）
- 显示平台标识和名称
- 提供"在新窗口打开"备用链接

#### Props 接口：

```typescript
interface VideoPlayerProps {
  url: string;          // 视频链接
  title?: string;       // 视频标题（可选）
  poster?: string;      // 视频封面图 URL（可选）
  className?: string;   // 自定义 CSS 类名（可选）
}
```

#### 使用示例：

```tsx
import VideoPlayer from '@/components/VideoPlayer';

<VideoPlayer
  url="https://www.bilibili.com/video/BV1xx411c7mD"
  title="腾讯 WXG 后端开发实习体验分享"
  className="mt-4"
/>
```

### 4. 视频链接检测器组件 (`VideoDetector`)

**功能**：自动检测文本中的视频链接并渲染播放器

**使用场景**：在问答科普中，如果 AI 回答包含视频链接，自动显示为视频播放器

**Props 接口**：

```typescript
interface VideoDetectorProps {
  content: string;      // 包含视频链接的文本
  className?: string;   // 自定义 CSS 类名（可选）
}
```

**使用示例**：

```tsx
import { VideoDetector } from '@/components/VideoPlayer';

<VideoDetector content={aiResponse} />
```
