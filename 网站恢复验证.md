## 🚨 如果遇到问题

### 问题1：端口3000被占用
**解决方案**：
```bash
# 停止占用端口的进程
netstat -ano | findstr :3000
taskkill /F /PID <进程ID>
```

### 问题2：页面无法加载
**解决方案**：
1. 检查终端，确认服务器正在运行
2. 清除浏览器缓存（Ctrl+Shift+Delete）
3. 强制刷新页面（Ctrl+F5）
4. 重启服务器（npm run dev）

### 问题3：JavaScript错误
**解决方案**：
1. 打开浏览器控制台（F12）
2. 查看错误信息
3. 如果看到模块找不到的错误，运行：
   ```bash
   rm -rf node_modules package-lock.json
   npm install
   ```

### 问题4：文件解析失败
**可能原因**：
- PDF是扫描版（图片），需要OCR转换
- Word文件是旧版.doc格式，建议转换为.docx
- 文件太大（>10MB），解析时间长

**解决方案**：
- 使用文本型PDF或.docx文件
- 减小文件大小
- 手动复制粘贴文本
